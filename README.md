# Análise de Dados com SQL — Sakila Database

Projeto de análise exploratória utilizando o banco de dados Sakila no MySQL.

## Objetivo
Extrair insights relevantes a partir de dados de uma locadora de filmes,
utilizando consultas SQL avançadas.

## Análises realizadas
- Top 10 filmes mais alugados
- Faturamento total por categoria de filme
- Top 10 clientes que mais gastaram
- Faturamento mensal

## Ferramentas utilizadas
- MySQL
- SQL (JOIN, GROUP BY, ORDER BY, HAVING, subqueries)
- MySQL Workbench

## Arquivos
- `analise_sakila.sql` — scripts com todas as consultas
- Prints dos resultados de cada análise
